ESO Rollplay - Guia do Mestre do Jogo (GM)

Começando

Tornar-se um GM é fácil - simplesmente abra suas Opções de seu Painel de Controle (ícone de chave inglesa) e defina a si mesmo como Mestre do Jogo. Assim que fizer isso, uma opção de Tela do GM irá aparecer em seu Painel de Controle.

A Tela do Mestre do Jogo

A Tela do Mestre do Jogo lhe permite fazer sessões do Rollplay por conta própria. Como GM, você será o árbitro das ações. Lembre-se que você sempre terá a palavra final. Você é livre para falsificar ou mesmo ignorar os resultados do dado se a situação permitir.

Controles da Sessão

  • Abrir Sessão Botão — Use este botão para iniciar uma nova sessão. Sua conta pode ter apenas uma sessão aberta ao mesmo tempo, iniciar uma nova sessão irá sobrescrever qualquer sessão anterior que estava acontecendo!
  • Finalizar Sessão Botão — Use este botão quando sua sessão estiver completa. Isso deixará a sessão como inativa e jogadores não terão mais permissão para se juntar. Quando a sessão estiver nesse estado, você ainda poderá vê-la a partir da Tela do Mestre do Jogo se você quiser revisar os eventos - contanto que você não tenha matado-a ao abrir outra sessão.

Controles dos NPCs

Abaixo da seção de NPCs, você poderá inserir o nome de um NPC e usar o botão Adicionar para adicioná-los à sessão. O NPC adicionado irá aparecer em uma lista ao lado esquerdo da seção. NPCs começam com um status de Oculto, o que significa que jogadores em sua sessão não poderão vê-lo. Você pode definir seus status para qualquer padrão de nível de ferimento para eles aparecerem para os jogadores. Por fim, você pode clicar no X em um registro de NPC na lista para remover esse NPC da sessão.

Clicar em um NPC da lista irá mostrar seus status no painel de cor mais clara ao lado direito (quando você adiciona um NPC, será selecionado automaticamente para isso). Nesse painel, você poderá definir os status do NPC. Esses status são muito mais simples do que os usados pelos personagens, porque o Rollplay apenas precisa saber suas capacidades de combate.

Controles do Jogador

Abaixo da seção de Jogadores você poderá inserir o nome de um jogador e usar o botão Adicionar para adicioná-los à sessão. Alternativamente, se você não tem certeza de como pronunciar o nome de um personagem, você pode usar o botão ? para procurar pelo nome do jogador. Assim como os NPCs, você poderá definir o status de ferimento ou usar o X para removê-los da sessão.

Além disso, criaturas invocadas serão mostradas abaixo dos jogadores quais elas pertencem. Assim como os jogadores, você poderá definir o status de ferimento ou usar o X para forçá-los a serem desconjurados.

Clicar em um jogador da lista irá mostrar seus status impressos ao lado direito no painel mais claro (quando você adiciona um jogador, será selecionado automaticamente para isso). Nesse painel, há também um link para visualizar a descrição do personagem. Abaixo do painel, opções de transformações irão aparecer se o personagem tem um tipo sobrenatural que é capaz disso - você pode forçar o jogador a iniciar ou finalizar a transformação.

Essas opções de transformações incluem uma opção secreta para lobisomens que apenas o GM tem permissão de selecionar para um jogador - Lobisomens Behemoth!

Atributos: +9 FOR, +1 AGI, +2 VEL, +5 RES
Bônus de Habilidades: Percepção +4; Sobrevivência +2; Desarmado +3.
Resistências: Doença
Fraquezas: Veneno, Prata

Histórico de Sessão

Esse painel mostra os eventos que transpiraram até então nas sessões do Rollplay. Comparado à tela de sessão do jogador do Rollplay, você poderá ver mais informações aqui. Eventos que apenas o GM pode ver serão mostrados com um background mais escuro.

Além disso, muitos eventos irão requerer que você faça a determinação de seu sucesso. Isso será feito com botões que aparecem diretamente no evento.

Controles de Rolagem

Esses são seus controles principais para executar uma sessão. No topo, você encontrara uma lista para selecionar um NPC. Essa seleção irá escolher qual NPC para usar em qualquer coisa que não tem a própria especialização selecionada. Também, há um campo de comentário que será usado para todas as rolagens nesse painel.

  1. Rolar! Botão — Este botão é usado para realizar rolagens simples, opcionalmente com um modificador selecionado da lista adjacente. Use isso para realizar rolagens que não interagem com outras coisas, como um NPC rolando contra Alquimia para misturar poções.
  2. Alvejar Jogador — Esta seção começa com uma lista que você pode usar para alvejar um jogador ou um mascote invocado. Esse alvo será usado para diferentes tipos de rolagens.
    • Atacar! Botão - Usando o NPC selecionado no topo do painel, inicie um ataque contra o jogador/mascote alvejado. Todas as rolagens do NPC serão feitas por você automaticamente.
    • Pronto para Rolar Botão - Isso manda uma rolagem para o jogador selecionado (não poderá ser selecionado para mascotes), fazendo-os rolar usando a habilidade ou atributo selecionado na lista vs. à direita.
    • Rolagem Contestada Botão - Isso manda uma rolagem para o jogador selecionado (não poderá ser selecionado para mascotes), fazendo-os rolar a habilidade ou atributo selecionado na lista vs. à direita. Esta rolagem é oposta à outras rolagens do NPC selecionado ao topo do painel, feita automaticamente por você, usando o modificador que você selecionar à esquerda do botão. Uma vez que os jogadores realizam suas rolagens, você poderá determinar se foi sucesso ou falha.
  3. Rolagens de Jogador contra Jogador — Essa seção possui várias listas para preparar dois jogadores para rolar habilidade/atributo um contra o outro. Você pode usar isso para dois jogadores que estiverem lutando ao fazê-los rolar Força contra Força, por exemplo. Note também que você deve selecionar dois jogadores diferentes para isso, você não pode fazer um personagem lutar contra si mesmo!
  4. Fazer uma Postagem de Mestre do Jogo — Essa seção é usada para transportar informação para jogadores que não pode ser feita facilmente através de rolagens. Por exemplo, se você fizer todos os jogadores da sessão rolarem Percepção contra seus NPCs ocultos, você pode usar uma Postagem de GM para dizer aos seus jogadores os resultados (por exemplo "Devlan, Skjolda e Orlug podem escutar passos leves e grunhidos baixos nas árvores em volta do campo.").

Lidando com Rolagens de Jogadores

Quando você manda uma rolagem para um jogador, tenha em mente que poderá levar um tempo para ele realizar sua rolagem. Se você tentar mandar uma rolagem e disser que o jogador está ocupado, isso significa que ele está atualmente usando o Pop-Up de Rolagem para fazer outra rolagem. Você precisará esperar aquela rolagem vir e tentar novamente.

Tenha certeza de prestar atenção aos eventos que estão por vir no Histórico de Sessão. Muitos deles requerem sua entrada antes de serem "completados". Idealmente, você não deve deixar qualquer evento sem resolver no feed do Histórico de Sessão.

Cuidado especial deve ser tomado quando processar rolagens de ataque. Quando você determina se a rolagem tem sucesso, ou falha, também precisará usar a descrição in-game do ataque para definir o tipo de dano. Tenha certeza de definir o tipo apropriado aqui para então interagir com resistência/fraqueza a dano corretamente.
