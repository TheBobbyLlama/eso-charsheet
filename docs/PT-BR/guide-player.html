ESO Rollplay - Guia do Jogador

Começando

Assim que você criar uma conta, você será levado para a tela do Painel de Controle onde você poderá criar seu primeiro personagem. Há um botão de Ajuda no topo da página de Ficha do Personagem que te ajudará a guiar-se por todo o processo. Quando fizer seu primeiro personagem, você poderá voltar ao Painel de Controle ao clicar no título ESO Rollplay no topo da tela.

De volta ao Painel de Controle, você verá uma lista de personagens que fez. Poderá clicar em qualquer um deles para visualizar sua ficha de status, assim como habilitar opções de Jogar, Editar ou Deletar o personagem. As opções de Deletar e Editar funcionam como esperado. A opção Jogar lhe permite que seu personagem se junte a uma sessão ativa do Rollplay. Você precisará de uma sessão definida pelo Mestre do Jogo para usar o resto desse guia!

A Sessão do Rollplay

Aqui é onde você participará nas mecânicas de rolagem do roleplay. Isso é um suplemento para as ações na tela, então poderá ser mais conveniente acessá-la de um dispositivo móvel ao em vez de seu PC.

  1. Status Local
  2. Histórico de Sessão
  3. Controles de Rolagem
  4. Pop-up de Rolagem

Status Local

Aqui é a seção da tela que está ao topo quando vista pelo celular, ou pela esquerda quando se vê pelo PC.

  1. Carregar Sessão Botão — Geralmente esse botão será desnecessário, mas se seu Mestre do Jogo ainda não tiver uma sessão preparada para você, irá receber um erro quando entrar nessa tela "Esse personagem não faz parte de uma sessão ativa de roleplay. Verifique com seu Mestre do Jogo." Você pode usar o botão de Carregar Sessão para tentar se juntar novamente.
  2. Seu Personagem — Uma simples leitura da ficha de seu personagem.
  3. Info de Personagem — Essa seção permitirá você de editar o atual status de seu personagem. Você poderá definir sua arma e armadura equipada aqui. Se seu personagem tiver um tipo sobrenatural capaz de se transformar, você poderá iniciar a transformação aqui também. Você precisará esperar o Mestre do Jogo confirmar sua transformação.
  4. Grupo — Essa seção mostra os jogadores no grupo e suas criaturas invocadas. As entradas são codificadas em cores de acordo com seu status de ferimento.

Histórico de Sessão

No dispositivo móvel, ou no canto direito superior quando no PC, há o painel de Histórico de Sessão. Ele mostra a você os eventos que transpiraram até agora nas sessões do Rollplay.

Controles de Rolagem

O último painel no dispositivo móvel, ou no canto direito inferior no PC, é para os controles de rolagem. Essa é a área principal que você irá usar para interagir com uma Sessão do Rollplay.

  1. Realize uma Rolagem! — É o que você usará com mais frequência para inciar rolagens feitas por você.
    • Usando Rolagem - Escolha o atributo ou habilidade que você estará usando para a rolagem que realizará. Essa lista é um tanto assustadora, mas é sortida para mostrar as habilidades que você investiu pontos primeiro.
    • Role! Botão - Esse botão irá iniciar uma rolagem contra seu atributo e habilidade escolhidos. Isso é o que você irá usar para fazer rolagens como Percepção contra inimigos ocultos, ou Força para mover objetos pesados.
    • Ataque! Botão - Esse botão irá iniciar um ataque contra o alvo que você selecionou de uma lista adjacente. Essa opção deve ser usada apenas quando a ação que você fez causou dano ao alvo. Se você quiser, usar Ilusão para confundir um inimigo, use o botão Role!
  2. Invocação — Essa seção é usada para invocações e criaturas controladas. Você pode ter apenas uma criatura invocada por vez.
    • Se nenhuma criatura foi invocada - Você pode usar a lista para selecionar o tipo de criatura que gostaria de invocar. Abaixo da lista a um campo para inserir o nome da criatura invocada, se você quiser providenciar um. Finalmente, o botão de Invocar! irá iniciar a invocação. Você precisará realizar uma rolagem dependendo do tipo de criatura invocada, e o Mestre do Jogo irá decidir se a invocação será um sucesso.
    • Quando uma criatura for invocada - Você pode direcionar a invocação para atacar um alvo usando o botão Atacar! e sua lista adjacente. Você também pode usar o botão Dispensar! quando você não quiser mais sua criatura invocada.
  3. Inimigos — Essa seção irá mostrar os atuais inimigos que seu grupo está atento. Assim como na lista de Grupo, as entradas são codificadas por cor de acordo com seu status de ferimento.
  4. Modo Preguiçoso — Não quer clicar no Pop-Up de Rolagem quando você precisar fazer uma rolagem? Sem problema! O Modo Preguiçoso irá completar automaticamente suas rolagens e enviá-las imediatamente.

Pop-Up de Rolagem

Você verá esse pop-up quando você precisará realizar uma rolagem. Isso irá providenciar uma pequena mensagem sobre o que você estiver rolando, assim como um comentário pelo GM se aplicável. Você pode inserir um comentário próprio antes de rolar. Se você iniciou a rolagem por si próprio, você também tem a opção de cancelar a rolagem.

Assim que você rolar, você verá o resultado para sua rolagem, e então um botão de Ok irá aparecer para você para completar sua rolagem e enviá-la pra o GM.
